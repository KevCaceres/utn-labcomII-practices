Inicio de sesión

En un archivo aparte, llamado login.html, crear la estructura básica de un documento HTML que servirá de página de inicio de sesión. La misma debe incluir: Un título (h1) descriptivo de la página. Una breve explicación de los pasos a seguir para iniciar sesión. Un formulario que incluya un campo de texto y un campo de clave. Un botón para enviar el formulario. Un checkbox para recordar los datos del usuario

Como ingresar con mi cuenta?

El formulario se utiliza para verificar las credenciales de usuario y dar ingreso.

El usuario debe completar los campos requeridos para despues cargar y enviar lo datos. El checkbox permite al usuario recordar sus datos la proxima vez que entre en dicha pagina web y pueda agilizar su ingreso a la pagina.
